# Zak

## Overview
Discovering new and interesting content can be a bit of a pain provided there
is so much stuff going on all the time (Yes, the internet is a crazy place).
Zak allows you to discover interesting content all in one place. Users can
register and login with facebook. Once they're logged in, they can create a new post ie share
links, comment on a new post, open the link the post points to and users can even upvote posts
so that you see only the best content upon opening the page.
